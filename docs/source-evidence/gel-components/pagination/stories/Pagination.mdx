## About this component

We use pagination to divide large volumes of content into pages.

Pagination allows a user to navigate to and through pages within a series . It provides context by showing a user:

- which page their viewing
- sequential page numbers
- the total number of pages

It also allows page content to load faster.

We locate pagination under the content of the page.

## Default

<Canvas sourceState='none'>
    <Story story={stories.Default} />
</Canvas>

Example on how to use the `currentPage` and `onPageChange` props and hooks to change the current pagination.
`onPageChange` callback returns the page number and an object containing first and last index which are calculated by the page you are on, it only runs when buttons have been clicked on.
If you need to do some logic on the first page on page load, make sure you set the `currentPage` to `1` which will re trigger the callback and let you handle your results.
If your `pageSize` prop is `10`, and the `currentPage` is `5` it will return first index `40` and last index `50` which woud indicate you to get results `40` to `50`.

## Setting a current page

Use the `currentPage` prop to set the pagination to a default page, it could be useful when navigating between tabs or pages.
