# Horizontal rule

<Subtitle>
    Horizontal line that will be used to separate content. Margin top and bottom
    can be configured to suit your needs.
</Subtitle>

## About this component

A horizontal rule is a visual element that creates visual hierarchy by separating content. Use horizontal rule sparingly. Unnecessary visual elements can be distracting and increase a user’s cognitive load.

The horizontal rule should serve a functional, not decorative purpose.

Consider using whitespace (or negative space) over horizontal rule, as it can accomplish the same goal. For example, when separating dense body content from other UI elements.
