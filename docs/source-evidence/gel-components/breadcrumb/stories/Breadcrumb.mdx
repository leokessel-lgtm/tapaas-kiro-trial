# Breadcrumbs

<Subtitle>
    Give users a sense of place within a website. They also help orient them
    through a hierarchical series of links.
</Subtitle>

## About this component

Breadcrumbs show the path from the homepage to the page a user is on.

When users arrive on a page from a search, breadcrumbs allow them to explore similar content by providing an easy link to the page's parent or grandparent pages.

## Best practices

<PanelTwoColumnContainer>
    <Panel title='Do' type={PanelType.SUCCESS}>
        <ul>
            <li>Use on pages 3 or more levels deep from the homepage.</li>
            <li>
                Start the breadcrumb with the homepage and end with the parent
                page.
            </li>
            <li>
                Exclude the current page. The page title already acts as a
                navigational signpost for users. It should not be duplicated on
                the screen.
            </li>
            <li>
                Place the breadcrumb between the global navigation and the page
                title.
            </li>
            <li>Match the page name in the breadcrumb.</li>
            <li>Keep page titles succinct to avoid long breadcrumbs.</li>
            <li>
                Display a breadcrumb that reflects the user's journey to the
                page.
            </li>
            <li>
                Display the most popular path for users who arrive directly at
                the page from a search.
            </li>
        </ul>
    </Panel>
    <Panel title="Don't" type={PanelType.ERROR}>
        <ul>
            <li>
                Show the path the user took to arrive at a page. Instead, use
                the page's hierarchical location.
            </li>
            <li>Rely on breadcrumbs as your only navigation.</li>
            <li>Use breadcrumbs on multi-step forms or wizards.</li>
            <li>Use breadcrumbs if an expanded sidebar is displayed.</li>
            <li>
                Use breadcrumbs when navigation tabs are active on the screen.
            </li>
        </ul>
    </Panel>
</PanelTwoColumnContainer>
