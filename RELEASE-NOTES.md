# GEL Kiro TaPaaS Trial Pack v0.3

## What changed

This version replaces the earlier v0.2 package as the active Kiro trial build pack.

New additions include:

- TaPaaS source inventory from the component and template `.fig` files
- component registry for the first 12 TaPaaS candidates
- page template registry for repeatable transaction skeletons
- Figma evidence notes and evidence log
- TaPaaS Kiro steering files
- trial-only `src/tapaas-preview/` composite components
- a non-production vehicle-style transaction skeleton covering Privacy, Search, Declaration, Review and Confirmation
- a second "Apply for a trial permit" skeleton as a prompt-repeatability proof
- a skeleton switcher in `src/App.tsx`
- removal of the old NDISWC example pages from the active package

## Boundary

This pack supports trial use by engineers. It does not claim production readiness, WCAG compliance, accessibility approval, content approval, legal approval, privacy approval, policy approval, TaPaaS engineering approval or GEL sign-off.

Teams must still review their own implementation before production use.
